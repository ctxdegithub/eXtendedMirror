## FEATURES ##
### General Features ##
- Compiler Independent
- Non intrusive
- No mandatory parser required
- Variant based interface
- Less possible code duplication

### Supported C++ Constructs ###
- Primitive Types
- Pointer Types
- References
- C Array Types
- Classes
- Multiple Inheritance
- Abstract Classes
- Methods
- Static Functions
- Namespaces
- Class Templates
- Constants
